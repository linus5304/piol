# Autonomous Mode

1. Study agent/spec.md thoroughly
2. Study agent/implementation_plan.md
3. Pick the highest leverage unchecked task
4. Complete the task
5. Write a test to verify (if applicable)
6. Check the box [x] in implementation_plan.md
7. Repeat until all boxes checked
8. Say "DONE - ready for review"

Use /commit when you have a logical unit of work.
Use "ship it" when all boxes are checked.
